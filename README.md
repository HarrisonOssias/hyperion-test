# hyperion-test

## How to Run React-app
1. `cd hyperion-test`
2. `cd client`
3. `yarn install --save`
4. `yarn start`



---

## How to Run Express Server
**Prep**
1. Create a .env file
2. Past in DATABASE_URL from email

**Open new terminal**
1. `cd server`
2. `yarn install --save`
3. `yarn start`
